# Zoom録画 → YouTube 自動アップロードツール

Googleスプレッドシートに記載されたZoom録画のダウンロードリンクを読み取り、指定のYouTubeチャンネルへ自動アップロードするツールです。

---

## このツールでできること・できないこと

### できること
- スプレッドシートのリンクからZoom録画を自動ダウンロード
- 指定したYouTubeチャンネルへ自動アップロード（限定公開）
- アップロード完了後、スプレッドシートにYouTube URLと「済」を自動記入
- 24時間以上処理中の動画を自動で再アップロード
- 毎月決まった日に自動実行するスケジュール設定が可能
- 複数のYouTubeチャンネルへの振り分けアップロード

### できないこと
- 動画の編集・変換（録画そのままアップロードされます）
- YouTubeへの動画説明文・タグの自動入力（空欄でアップロードされます）
- Zoom以外のサービスからのダウンロード（直接ダウンロードリンクがあれば対応可能な場合あり）
- パソコンの電源が切れている間の自動実行

| Google Sheets API | 無料 | |
| このスクリプト | 無料 | オープンソース |

### YouTube APIの無料上限について

YouTube APIには1日あたりの利用上限（クォータ）があります。

- **1日の無料上限**：10,000クォータ
- **動画1本のアップロード**：約1,600クォータ消費
- **1日にアップロードできる目安**：約6本

このツールはデフォルトで1回の実行につき最大5本までに制限しており、無料枠を超えないようにしています。

### アップロード数を増やしたい場合

**方法A：Google APIの審査申請（推奨）**
- Googleにアプリの審査を申請すると、上限が撤廃されます
- 審査には数日〜数週間かかります
- 費用は無料です

**方法B：Google Cloudプロジェクトを複数作成する**
- プロジェクトを追加で作成するごとに10,000クォータが追加されます
- 設定の手間はかかりますが、費用は無料です

## セットアップ手順

### STEP 1：Pythonのインストール確認

Pythonはこのツールを動かすために必要なソフトです。

1. **ターミナルを開く**
   - Macの場合：`Launchpad` → `その他` → `ターミナル`
   - または `Command + スペース` で「ターミナル」と検索

2. **以下のコマンドを入力してEnterキーを押す**
   ```
   python3 --version
   ```

3. **結果の確認**
   - `Python 3.10.x` 以上が表示されればOKです
   - `command not found` と表示された場合は [python.org](https://www.python.org/downloads/) からインストールしてください

---

### STEP 2：ファイルの配置

1. このページの緑色の「**Code**」ボタンをクリック
2. 「**Download ZIP**」を選択してダウンロード
3. ZIPファイルを解凍する
4. 解凍したフォルダの中にある以下のファイルを確認する
   - `upload.py`
   - `config.example.json`
   - `.gitignore`

5. ターミナルで以下を実行して保存フォルダを作成する
   ```
   mkdir -p ~/Documents/movie-upload
   ```

6. 上記のファイルを `~/Documents/movie-upload/` フォルダに移動する

---

### STEP 3：必要なライブラリのインストール

ターミナルで以下のコマンドを入力してEnterキーを押してください。

```
pip3 install google-api-python-client google-auth-oauthlib openpyxl requests
```

`Successfully installed` と表示されれば完了です。

---

### STEP 4：Google Cloud プロジェクトの作成

GoogleのAPIを使うために必要な設定です。費用はかかりません。

#### 4-1. プロジェクトを作成する

1. ブラウザで [Google Cloud Console](https://console.cloud.google.com/) を開く
2. Googleアカウントでログインする
3. 画面上部のプロジェクト名をクリック → 「**新しいプロジェクト**」をクリック
4. プロジェクト名に `movie-file-upload` と入力 → 「**作成**」をクリック

#### 4-2. YouTube Data API を有効化する

1. [このリンク](https://console.cloud.google.com/apis/library/youtube.googleapis.com) を開く
2. 画面上部でさきほど作ったプロジェクト `movie-file-upload` が選択されていることを確認
3. 「**有効にする**」ボタンをクリック

#### 4-3. Google Sheets API を有効化する

1. [このリンク](https://console.cloud.google.com/apis/library/sheets.googleapis.com) を開く
2. 「**有効にする**」ボタンをクリック

---

### STEP 5：OAuth 認証情報の作成

このツールがGoogleのサービスにアクセスするための「鍵」を作ります。

#### 5-1. OAuth 同意画面を設定する

1. [認証情報ページ](https://console.cloud.google.com/apis/credentials) を開く
2. 画面右上の「**同意画面を構成**」または左メニューの「**OAuth 同意画面**」をクリック
3. User Type で「**外部**」を選択 → 「**作成**」をクリック
4. 以下を入力する
   - **アプリ名**：`Movie Upload Tool`
   - **ユーザーサポートメール**：自分のGmailアドレスを選択
5. 「**次へ**」をクリック

6. スコープの画面で「**スコープを追加または削除**」をクリックし、以下の2つを追加する
   - `https://www.googleapis.com/auth/youtube`
   - `https://www.googleapis.com/auth/spreadsheets`
   
   （検索ボックスに `youtube` や `sheets` と入力すると見つけやすい）

7. 「**更新**」→「**保存して次へ**」をクリック

8. テストユーザーの画面で「**ADD USERS**」をクリックし、アップロードに使うすべてのGmailアドレスを追加する

9. 「**保存**」をクリック

#### 5-2. OAuth クライアントIDを作成する

1. 左メニューの「**認証情報**」をクリック
2. 「**＋ 認証情報を作成**」→「**OAuth クライアント ID**」をクリック
3. アプリケーションの種類で「**デスクトップ アプリ**」を選択
4. 名前に `movie-upload-client` と入力 → 「**作成**」をクリック
5. ポップアップが表示されたら「**JSONをダウンロード**」をクリック
6. ダウンロードしたファイルを **`client_secrets.json`** にリネームする
7. `client_secrets.json` を `~/Documents/movie-upload/` フォルダに移動する

---

### STEP 6：スプレッドシートの準備

以下の列構成でスプレッドシートを用意してください。

| 列 | 項目名 | 内容 |
|---|---|---|
| A | 日付 | 録画日 |
| B | チーム | チーム名など |
| C | 録画担当 | 担当者名 |
| D | 共有リンク | Zoomの共有URL |
| **E** | **ダウンロードリンク** | **Zoomのダウンロード用URL** |
| F | YouTubeリンク | アップロード後に自動入力される |
| G | 備考 | メモなど |
| H | 日付（タイトル用） | YouTubeタイトルに使う日付 |
| **I** | **動画タイトル** | **YouTubeのタイトルになる（空欄はスキップ）** |
| J | チェック | アップロード後に「済」が自動入力される |
| **K** | **タグ** | **アップロード先チャンネルを選ぶドロップダウン** |
| L | アップロード日時 | 自動入力（触らなくてOK） |

> **ポイント**：列の順番は設定ウィザードで変更できます。上記は推奨構成です。

---

### STEP 7：設定ウィザードの実行

ターミナルで以下を実行し、画面の指示に従って入力してください。

```
python3 ~/Documents/movie-upload/upload.py config
```

入力が必要な内容：
- スプレッドシートのURL
- 各列の番号（Aが1、Bが2、Cが3…）
- YouTubeチャンネルの数と名前

完了すると `config.json` が自動で作成されます。

---

### STEP 8：認証セットアップ

各YouTubeチャンネルへのアクセス権を設定します。チャンネルの数だけブラウザが開くので、それぞれのアカウントでログインしてください。

```
python3 ~/Documents/movie-upload/upload.py setup
```

> **注意**：ブラウザに「このアプリはGoogleで確認されていません」と表示された場合は、「詳細」→「安全ではないページに移動」をクリックして続行してください。

---

### STEP 9：タグのドロップダウンを追加

スプレッドシートのK列にチャンネル選択用のドロップダウンを追加します。

```
python3 ~/Documents/movie-upload/upload.py sheet
```

スプレッドシートを開いてK列にドロップダウンが追加されていれば成功です。

## 使い方

### 手動で実行する

ターミナルで以下を実行します。

```
python3 ~/Documents/movie-upload/upload.py
```

実行前にスプレッドシートのK列にタグを入力しておいてください。

### 毎月自動で実行する（例：毎月5・10・15・20・25・30日 朝9時）

以下のコマンドを1回だけ実行すれば、以降は自動で動きます。

```
(crontab -l 2>/dev/null; echo "0 9 5,10,15,20,25,30 * * /usr/bin/python3 ~/Documents/movie-upload/upload.py >> ~/Documents/movie-upload/log.txt 2>&1") | crontab -
```

> **実行日は自由に変更できます。**
> 上記コマンドの `5,10,15,20,25,30` の部分を変えることで実行日を調整できます。
> 例：毎月1日だけ実行したい場合は `5,10,15,20,25,30` を `1` に変更してください。
> 時刻を変えたい場合は `0 9` の `9` を変更します（例：`0 18` で18時実行）。

設定の確認：
```
crontab -l
```

## よくある質問

**Q. 「このアプリはGoogleで確認されていません」と表示される**
A. 個人利用のアプリのため正常です。「詳細」→「安全ではないページに移動」をクリックして続行してください。

**Q. アップロードが途中で止まった**
A. 次回実行時に自動的に再開されます。24時間以上処理中の場合は自動で再アップロードされます。

**Q. 1日に何件アップロードできる？**
A. 無料枠では1日最大6件程度です。`config.json` の `max_uploads_per_run` で上限を変更できます。

**Q. 動画はパソコンに保存される？**
A. 保存されません。一時的にダウンロードしてYouTubeにアップロードした後、自動で削除されます。
